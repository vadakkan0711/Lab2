# Breast Cancer Classification using Machine Learning

## Overview
This project builds a **Breast Cancer Classification Model** using **Logistic Regression** and **Random Forest Classifier**. The model predicts whether a tumor is **Malignant (M)** or **Benign (B)** based on medical features.

## Dataset
- **Dataset Name:** Breast Cancer Wisconsin (Diagnostic)
- **Source:** UCI Machine Learning Repository
- **Features:** 30 numerical features extracted from digitized breast cancer images
- **Target:** `Diagnosis` (M = Malignant, B = Benign)
- **Size:** 569 samples
